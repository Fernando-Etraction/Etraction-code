# ArtFactory — seções separadas

Cada pasta contém:

- `section.html`: HTML isolado da seção, já dentro do wrapper `.fernando-cro-artfactory-lp`.
- `section.css`: CSS necessário para a seção, incluindo base global, botões, tipografia e responsivo.
- `section.js`: JavaScript necessário para a seção. Se a seção não tiver interação específica, mantém apenas reveal/scroll quando aplicável.
- `index.html`: preview individual da seção.

Arquivos na raiz:

- `preview-completo.html`: página completa recomposta.
- `styles.css`: CSS original completo.
- `script.js`: JavaScript original completo.

Se for colar uma seção em uma plataforma, use juntos o `section.html`, `section.css` e `section.js` da respectiva pasta.
