# LP Whats - seções separadas

Cada pasta contém uma seção isolada:

- `section.html`: apenas o HTML da seção.
- `section.css`: CSS necessário para a seção funcionar visualmente de forma independente.
- `section.js`: JavaScript da seção, quando aplicável.
- `index.html`: preview da seção usando CSS e JS separados.
- `section-completo.html`: HTML único com CSS e JS embutidos no mesmo arquivo.

O arquivo `preview-completo.html` mantém a landing page original completa.
